Guide d'Utilisation
==================

Structure des Données
-------------------
Le projet utilise des données de différents capteurs:

* PS1-PS6: Pression (100 Hz)
* EPS1: Puissance (100 Hz)
* FS1-FS2: Débit (10 Hz)
* TS1-TS4: Température (1 Hz)
* VS1: Vibrations (1 Hz)
* CE, SE: Efficacité (1 Hz)
