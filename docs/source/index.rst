Analyse Prédictive des Systèmes Hydrauliques
===========================================

Ce projet propose une analyse avancée des séries temporelles pour surveiller et prédire l’efficacité des systèmes hydrauliques à l’aide de modèles d’apprentissage automatique (LSTM, GRU, MLP). Une application interactive Streamlit permet de visualiser les données, prédire l’efficacité système (SE), et optimiser les paramètres des capteurs.
